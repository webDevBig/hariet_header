Image positioning and sizing classes for c-block-05__media

no class image takes up 100% of column's width, horizontally aligned to middle of row

Positioning:
.c-block-05__media--centered image is centered with a max width for the parent container of 460px (home page images with shadow, phone screenshot)

Sizing:
.c-block-05__media--width-md max image width is 460px
.c-block-05__media--height-sm max image height is 370px
.c-block-05__media--height-md max image height is 530px (same as on blocks with coloured backgrounds)

.c-block-01 widgets can be replaced with these, HTML is optimized to fit more use cases here.
